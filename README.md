Built a systematic return-forecasting LightGBM model for equity ETFs using end-of-day options data (>1M contracts/year), engineering signals capturing market sentiment (put–call ratios, flow), volatility risk premia (IV–RV spread), skew, term structure, and dealer gamma exposure.

Generated leverage-controlled trading signals (−1.0x to +1.5x), with robust out-of-sample performance (Calmar Ratio > 2.0).
